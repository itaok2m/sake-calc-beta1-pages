日本酒醸造 計算ツール

ホーム画面から開けるようにする

Android Chrome では、ホーム画面に追加すると次からアプリのように開けます。

Android Chrome の右上メニューから「ホーム画面に追加」または「アプリをインストール」を選んでください。

アルコール分温度換算表

測定度数と温度を入れて、下の資料の位置を合わせて確認します。

→

2mm表

尺棒で測った検尺値や目標数量から、容器検定簿と採用計算方式をもとに再生成した2mm表で容量と候補mmを確認します。

→

割水計算

原液の数量や成分をもとに、目標値へ合わせるための加水量や調整後の値を計算します。

→

混合計算

普段の2種混合を中心に、必要な時だけ3〜5種混合へ切り替えて計算できます。

→

アルコール添加計算

添加前後の数量や成分をもとに、必要なアルコール量や添加後の状態を計算します。

→

水温温度計算

2種類の水を混ぜる際の水温・水量を計算します。混合後の水温、追加水の条件、配合量の3パターンに対応します。

→

酛仕込温度計算

酛の蒸米温度を一覧表示します。

→

醪仕込温度計算

醪の蒸米温度を一覧表示します。

→

エキス分計算

エキス分と原エキス分を計算します。

→
